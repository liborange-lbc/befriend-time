## 新增需求

### 需求:用户可以查看均值偏差历史趋势
系统必须展示选定指数的均值偏差历史走势图。

#### 场景:显示偏差趋势图
- **当** 用户在数据分析页面切换到"均值偏差分析"Tab
- **那么** 系统显示该指数各周期偏差的历史趋势折线图
- **当** 用户选择时间范围
- **那么** 图表更新为对应时间段的数据

### 需求:用户可以查看偏差分布柱状图
系统必须展示选定周期内所有日期偏差的分布柱状图。

#### 场景:显示分布柱状图
- **当** 用户查看均值偏差分析
- **那么** 系统显示各周期的偏差分布柱状图（30/60/90/180/240/360日）
- **当** 柱状图渲染完成
- **那么** X轴显示偏差区间，Y轴显示该区间内的天数

### 需求:分布柱状图必须使用20个区间
系统必须将偏差分布划分为固定20个区间。

#### 场景:区间划分
- **当** 系统渲染偏差分布图
- **那么** 系统根据选定周期内所有偏差数据，划分为20个等宽区间
- **当** 数据范围变化
- **那么** 区间边界自动调整以覆盖所有数据

### 需求:分布柱状图必须标注当前位置
系统必须在分布柱状图上标注当前偏差值的位置。

#### 场景:当前位置标注
- **当** 分布柱状图渲染完成
- **那么** 系统在X轴相应位置绘制一条竖线，标注当前偏差值
- **当** 当前偏差值落在某个区间内
- **那么** 该区间柱子高亮显示或在竖线旁显示数值标签

### 需求:用户可以对比多周期偏差分布
系统必须在同一页面展示所有周期的偏差分布图。

#### 场景:多周期分布展示
- **当** 用户查看均值偏差分析
- **那么** 系统纵向排列展示 30/60/90/180/240/360 日周期的偏差分布图
- **当** 用户滚动页面
- **那么** 可以快速对比不同周期的偏差分布

### 需求:分布图必须紧凑展示
系统必须将分布图的名称和横轴在同一行展示。

#### 场景:紧凑布局
- **当** 系统展示多周期分布图
- **那么** 每行包含：周期名称 + 分布柱状图横轴 + 当前分位值
- **当** 多个分布图纵向排列
- **那么** 保持对齐，便于快速对比
