## 新增需求

### 需求:用户可以查看收盘价趋势图
系统必须展示选定指数的收盘价历史走势图。

#### 场景:显示收盘价图表
- **当** 用户在数据分析页面选择一个指数
- **那么** 系统显示该指数的收盘价折线图
- **当** 用户选择时间范围
- **那么** 图表更新为对应时间段的数据

### 需求:图表必须支持双Y轴
系统必须支持在同一图表上显示收盘价和均线，使用双Y轴区分。

#### 场景:双Y轴显示
- **当** 用户查看收盘价与均线叠加图表
- **那么** 左侧Y轴显示收盘价刻度，右侧Y轴显示均值刻度
- **当** 数据范围差异较大
- **那么** 双Y轴独立缩放，保证两条曲线都在可视范围内

### 需求:图表必须显示多周期均线
系统必须在收盘价图表上叠加显示各周期SMA均线。

#### 场景:均线叠加显示
- **当** 用户查看收盘价图表
- **那么** 系统默认显示 60 日SMA均线
- **当** 用户选择其他周期
- **那么** 图表叠加显示所选周期的SMA均线，不同周期使用不同颜色

### 需求:用户可以快速选择时间范围
系统必须提供时间范围快捷选择功能。

#### 场景:快选时间范围
- **当** 用户点击时间快选按钮
- **那么** 系统提供选项：近1年、近3年、近5年、近10年
- **当** 用户选择某个快选项
- **那么** 图表更新为对应时间范围的数据

### 需求:用户可以拖拽选择时间范围
系统必须支持通过拖拽图表区域选择时间范围。

#### 场景:拖拽选时
- **当** 用户在图表底部 dataZoom 区域拖拽
- **那么** 图表实时更新显示选中的时间范围
- **当** 用户放大或缩小选区
- **那么** 图表时间范围相应调整

### 需求:用户可以切换分析类型
系统必须支持通过Tab切换不同的分析类型。

#### 场景:Tab切换
- **当** 用户在数据分析页面点击不同Tab
- **那么** 系统切换显示对应的图表：收盘价分析、均值偏差分析
- **当** 切换Tab时
- **那么** 保持当前选中的指数和时间范围

### 需求:图表必须支持SVG导出
系统必须支持将图表导出为SVG格式图片。

#### 场景:导出SVG
- **当** 用户点击"导出图表"按钮
- **那么** 系统将当前图表导出为SVG文件
- **当** 导出完成
- **那么** 浏览器自动下载文件
